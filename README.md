### How to Run:
1. Clone the repo:  
   ```bash
   git clone https://github.com/rekcah-pavi/Flutter-Login-Page
   ```
2. Install dependencies:  
   ```bash
   flutter pub get
   ```
3. Run the app:  
   ```bash
   flutter run
   ```
